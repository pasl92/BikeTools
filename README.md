# BikeTools - WSEI 2019/2020

Aplikacja konsolowa C# przeznaczona dla rowerzystów

### Projekt oferuje następujące funkcjonalności:


##### - Kalkulator rozmiaru ram szosowych oraz MTB

Umożliwia dobranie odpowiedniej wielkości ramy rowerowej na podstawie podanego wzrostu.


##### - Kalkulator czasu jazdy na rowerze typu Ebike

Na podstawie liczny danych (amperogodzin, napięcia i mocy), funkcjonalność poda szacunkowy czas jazdy na jednej baterii. Wyliczenia są strikte teoretyczne i realne osiągi mogą być zależne od czynników takich jak waga, warunki wietrzne czy ukształtowanie terenu. Dane potrzebne do obliczeń znajdziemy na bateri lub internecie.

##### - Kalkulator BMI
Bmi obliczane na podstawie wagi i wzrostu.
